HIGH PERFORMANCE COMPUTING

SYLLABUS:

UNIT-I

Introduction to Parallel hardware and software, need for high performance systems and Parallel Programming, SISD, SIMD, MISD, MIMD models, Performance issues.

UNIT - II

PThreads, Thread Creation, Passing arguments to Thread function, Simple matrix multiplication using Pthreads.

UNIT - III

Pthreads: Critical sections, mutexes, semaphores, barriers and conditional variables, locks, thread safety, simple programming assignments.

UNIT - IV

Open MP Programming: introduction, reduction clause, parallel for-loop scheduling, atomic directive, critical sections and locks, private directive, Programming assignments, n body solvers using openMP.

UNIT - V

Introduction to MPI programming: MPI primitives such as MPI_Send, MPI-Recv, MPI_Init, MPI-Finalize, etc., Application of MPI to Trepizoidal rule, Parallel Quick sorting algorithm
